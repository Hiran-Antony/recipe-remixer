# Recipe Remixer
AI-powered recipe generator built with 
Next.js, MongoDB, and Groq LLM.

## Live Demo
https://recipe-remixer.vercel.app

## Features
- Real AI recipe generation (Groq/Llama3)
- User authentication (NextAuth)
- Save recipes to MongoDB
- Ingredient quick-add chips
- Mobile responsive design

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- MongoDB Atlas + Mongoose
- NextAuth.js
- Groq SDK (llama3-70b-8192)
- Tailwind CSS
- Framer Motion
